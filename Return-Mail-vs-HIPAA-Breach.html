HIPAA BREACH VS RETURN MAIL

What is a Return Mail concern?

  • This happens when a person receives a bill that is not intended to them and showing that the address on the guarantor account is incorrect.

Sample Scenario:

  • If you receive a phone call from patient stating that receive a bill that is address to a different patient then this is cconsidered as a Return Mail issue.

What should we do then?

  • Agent should advise the caller to either return or destroy the bill received, then advice will send an alert to the team to update the guarantors address.
  • Agent should add a RETURN MAIL STATUS and STATEMENT HOLD alert on the account.
    • RETURN MAIL STATUS - this will alert the offline team that the account has a wrong address on file.
    • STATEMENT HOLD Alert - would place the balance on hold and bills will not be mailed out until the address has been corrected and the RETURN MAIL status will be remove on the account.

What is HIPAA BREACH?

  • It is the acquisition, access, use, or disclosure of protected health information in a manner not permitted by the patient which compromises the security or privacy of their protected health information.

Sample Scenario:

  • If you received a call from patient stating they receive a bill that is address under their name containing charge for other patient that they do not know at all - this is HIPAA BREACH.

What should we do then?

  • Agent must inform their support with the complete details and supports will send an email to report it.

How to add RETURN MAIL STATUS and Statement hold Alert on the guarantors account

  1. Under Guarantor Activity select ACCOUNT STATUS CHANGE [585]
  2. Put notes on the Guarantor note and under Update Guarantor Status add RETURN MAIL.
  3. Remove the SET REMINDER by clicking the X at the upper right corner.
  4. Then Click on Accept
  5. Then the Status RETURNED MAIL will appear on the Guar account.
  6. After adding the status, we need to add the billing status to hold the statements Alert.
  7. On the Guarantor Edit Tab - go to FINANCE - select Age and Hold Stmts.
  8. Once done, the alert will appear on the Summary tab.
